Credit Risk Classification Using Machine Learning

##  Project Overview
This project predicts the credit risk of borrowers using machine learning models.
It helps financial institutions identify loan defaulters and make better lending decisions.

##  Dataset Description
The dataset includes borrower information such as age, income, loan amount,
interest rate, employment length, credit grade, and past default history.

##  Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Google Colab

##  Machine Learning Models
- Logistic Regression
- Random Forest Classifier
- Support Vector Machine
- K-Nearest Neighbors
- Naive Bayes

## Results
Random Forest Classifier achieved the highest accuracy and performed best.

## Target Variable
- loan_status (0 = Non-default, 1 = Default)

##  Conclusion
Machine learning can automate credit risk assessment and improve loan approval decisions.
